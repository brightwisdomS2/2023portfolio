skill

웹 표준과 웹 접근성을 준수하는 HTML 정보 구조설계로 웹사이트 퍼블리싱을 하고 있습니다. 특히 웹 표준을 위해 HTML5에서 권장하는 시맨틱 태그를 사용해 전체 레이아웃을 설계합니다. 웹 UI디자인 작업 후 클래스 네임과 함께 HTML 정보 구조설계를 꼼꼼히 작업한 후 퍼블리싱 작업을 진행합니다. 웹사이트를 퍼블리싱 하기 전에 ‘기능정의서’를 만들고 기능정의서에 전체적인 폰트 서식, 컬러 가이드, 사이즈 정보, HTML 와이어프레임, 구현되어야 할 기능들까지 꼼꼼하게 미리 정리해 놓고 체계적인 퍼블리싱 작업을 수행합니다.

CSS 코딩은 IE를 지원도 충분히 고려하는 크로스브라우징(Cross Browsing)을 위해 안정적인 CSS 속성을 사용해서 HTML Elements를 배치하고 디자인합니다. 특별한 경우 CSS3에서 새롭게 지원하는 속성을 활용해서 좀 더 다양한 CSS 배치와 디자인을 합니다.

네비게이션, 탭 메뉴, 어코디언 등 웹 퍼블리싱에 반드시 필요한 상호작용을 위한 제이쿼리는 플러그인을 사용하지 않고 순수한 코딩으로 제작합니다. 필수적인 자바스크립트 활용은 가능하며 주로 제이쿼리와 병행해서 활용합니다. (if 조건문, for 반복문 등)

how

Responsive Web

반응형 웹 제작에 있어서 크로스브라우징(Cross Browsing)을 위해 안정적인 기존 방식인 float, position, Media Query를 활용한 반응형 웹 제작 경험을 충분히 가지고 있습니다. 필요에 따라 플렉스(Flex)와 그리드(Grid) 방식으로 반응형 웹 사이트 제작도 가능합니다. 대부분 기존 방식으로 제작하지만 모바일 웹 앱 제작의 경우 Flex 또는 Grid를 충분히 활용해서 제작합니다.

Mobile Web App

모바일 우선을 위해 Mobile First에 준하는 모바일 웹 앱 제작 경험도 가지고 있습니다. UI/UX 측면에서 미적인 디자인 뿐만 아니라 모바일 사용자 경험의 최적화를 위해 효율적인 컨텐츠를 배치를 우선적으로 생각하며 화면을 구성합니다.

tool

웹사이트 UI 디자인 작업은 포토샵과 피그마, xd를 활용하며 웹 UI 디자인 작업을 위한 능력은 충분하다고 생각합니다. 또한, 홈페이지에 필요한 부가적인 디자인 요소들을 제작하기 위한 포토샵과 일러스트 사용에 익숙합니다. 컷편집, 자막, 트랜지션, 기본적인 모션그래픽 등 기본적인 영상편집 및 제작은 프리미어를 활용합니다.

텍스트에디터는 Visual Studio Code를 메인으로 사용하며 경우에 따라 Adobe Brackets를 병행해서 사용합니다. 팀원들과의 효율적인 작업을 위해 Git/GitHub도 공부하고 있습니다.
